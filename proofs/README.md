# PROOFS — gerações reais sob a mente

Cada geração vive em `gen-NNN/`:

| Artefato | Autor | O que é |
|---|---|---|
| `BRIEF.md` | maker | Design Brief completo (9 seções do CONTRACT, rastreável) |
| `GATE-VERDICT.md` | juiz frio (agente separado) | os 12 itens com evidência citada |
| `tokens.json` | coder | tokens W3C DTCG da identidade gerada |
| `core.css` | coder | o CSS core da identidade |
| `specimen.html` | coder | a prova viva com DADOS REAIS do produto |
| `lint.mjs` | coder | regras grepáveis derivadas do brief desta identidade |

Uma proof só vira preset (`presets/`) depois do veredito do diretor (`verdicts/`). Proof reprovada permanece no repo — é dado, não lixo.

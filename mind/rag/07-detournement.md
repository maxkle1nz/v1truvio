# RAG_CARD — Détournement Situacionista

- **nome:** Détournement Situacionista
- **domínio:** crítica cultural, apropriação, ressignificação
- **lei extraída:** signos comerciais podem ser desviados para restaurar consciência — a superfície familiar vira ferramenta quando seu significado é reencaixado
- **usar quando:** uma superfície familiar precisa ser transformada em ferramenta crítica ou operacional
- **traduzir em:** checkout como ritual, login como pórtico, dashboard como mesa de comando
- **interação:** quebrar fluxo automático onde há decisão relevante
- **motion:** interrupção elegante, não pop-up agressivo
- **som:** pequeno corte de atenção
- **evitar:** ironia excessiva, interface panfletária
- **risco de clichê:** culture jamming adolescente
- **componentes derivados:** ritual de entrada, etapa de acoplamento, confirmação de autoria
- **pareia bem com:** Stuart Hall (a teoria do desvio), Lissitzky (a forma do desvio)

**Pergunta do órgão (Memória Geracional, em modo subversivo):** *que padrão automático merece ser desviado aqui?*
